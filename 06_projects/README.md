# 第六阶段：项目实战

这一阶段不再以“单文件教学脚本”为主，而是切换到更接近真实工程的项目结构。

## 项目列表

1. `01_image_classification`：图像分类实战
2. `02_text_classification`：文本分类实战
3. `03_multimodal_retrieval`：图文检索/多模态对齐
4. `04_generative_lab`：生成模型实验台

## 统一目标

- 训练入口与评估入口分离
- 模型、数据、配置拆分
- 支持输出目录管理
- 每个项目都可逐步扩展到真实数据集

## 推荐推进顺序

1. 先完成 `01_image_classification`
2. 再补 `02_text_classification`
3. 然后做 `03_multimodal_retrieval`
4. 最后整理 `04_generative_lab`
