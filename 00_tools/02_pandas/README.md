# 02_pandas

目标：掌握真实业务中最常用的数据清洗和分析流程。

建议脚本数：10

建议文件：
1. 00_io_and_schema.py: CSV/Excel/Parquet 读取与类型检查
2. 01_missing_values.py: 缺失值填充和删除策略
3. 02_outlier_handling.py: 异常值识别和修正
4. 03_groupby_agg.py: 分组统计与多聚合
5. 04_merge_join.py: 多表 join 和键冲突处理
6. 05_pivot_reshape.py: pivot/melt/wide-long 转换
7. 06_datetime_ops.py: 时间序列和滚动窗口
8. 07_feature_engineering.py: 常用特征构造
9. 08_eda_report.py: 基础 EDA 报告
10. 09_pandas_mini_project.py: 端到端清洗+分析

学习重点：
- 数据质量检查流程
- join 后数据行数变化的解释
- 可读可维护的数据处理代码风格

验收标准：
- 给定中等规模表格数据，30-60 分钟可出分析结论
