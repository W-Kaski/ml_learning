# 04_sklearn

目标：补齐传统机器学习闭环能力。

建议脚本数：12

建议文件：
1. 00_baseline_classification.py: 逻辑回归基线
2. 01_baseline_regression.py: 线性回归基线
3. 02_preprocessing_pipeline.py: StandardScaler + Pipeline
4. 03_feature_selection.py: 特征筛选
5. 04_cross_validation.py: KFold/StratifiedKFold
6. 05_hyperparameter_search.py: GridSearch/RandomizedSearch
7. 06_metrics_classification.py: Precision/Recall/F1/AUC
8. 07_metrics_regression.py: MAE/MSE/R2
9. 08_model_comparison.py: 多模型对比
10. 09_error_analysis.py: 错误样本分析
11. 10_classification_project.py: 分类小项目
12. 11_regression_project.py: 回归小项目

学习重点：
- Pipeline 规范化建模
- 交叉验证防止数据泄漏
- 指标和业务目标对齐

验收标准：
- 能完成一个可复现的 sklearn 端到端项目
