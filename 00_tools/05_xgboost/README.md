# 05_xgboost

目标：掌握工业常用强基线模型与调参方式。

建议脚本数：7

建议文件：
1. 00_xgb_baseline.py: 分类/回归基线
2. 01_params_core.py: depth, eta, subsample 核心参数
3. 02_early_stopping.py: 提前停止与验证集
4. 03_imbalance_handling.py: 类别不平衡处理
5. 04_feature_importance.py: 特征重要性分析
6. 05_tuning_workflow.py: 调参工作流
7. 06_xgb_project_report.py: 小项目与结论

学习重点：
- 参数影响机制
- 过拟合诊断
- 和 sklearn 模型对比

验收标准：
- 你能给出清晰的调参路径和收益说明
