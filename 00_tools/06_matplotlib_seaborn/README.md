# 06_matplotlib_seaborn

目标：掌握模型分析和汇报最常用可视化。

建议脚本数：8

建议文件：
1. 00_plot_basics.py: figure/axes/样式管理
2. 01_distribution_plots.py: 直方图/KDE/箱线图
3. 02_correlation_heatmap.py: 相关性热力图
4. 03_time_series_plots.py: 时间序列可视化
5. 04_model_metrics_plots.py: ROC/PR/阈值分析
6. 05_confusion_matrix_plot.py: 混淆矩阵可视化
7. 06_error_analysis_dashboard.py: 错误分析图表集合
8. 07_visual_report_template.py: 可复用报告模板

学习重点：
- 图表表达准确性
- 配色与可读性
- 用图支撑结论

验收标准：
- 能输出一份完整模型评估图表报告
